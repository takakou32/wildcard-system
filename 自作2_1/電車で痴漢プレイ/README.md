## 注意事項

1. **連番の連続性**: NSFWプロンプトはSFWの最終連番+1から開始され、ファイル名順でシームレスに並びます
2. **服装の継続性**: 同じシーン内でSFWからNSFWへ移行する際、服装はビジネスウェア（`outfit_business`）で継続されます
3. **場所情報**: NSFWプロンプトには場所情報（crowded train, station at night等）が含まれています
4. **電車痴漢プレイ**: Scene 03では専用の`nsfw_chikan_all`を使用し、電車内での痴漢プレイを表現します
