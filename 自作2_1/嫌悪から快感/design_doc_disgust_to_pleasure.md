# 全フェーズ(嫌悪〜フィニッシュ〜事後) 詳細設計・プロンプト指示書

本ドキュメントは、各シーンにおける演出意図、カメラアングル・構図、ポーズ、表情の詳細な指示文をすべて「日本語」でまとめた設計書です。
今後のプロンプト作成・派生ワークフローでは、本ドキュメントの日本語指示を意味的に解釈し、`prompt_library.md` や `danbooru.csv` などを検索リソースとして活用することで、動的に英単語（Danbooruタグ化）への落とし込みを行います。

  - 男性: 強引に迫り、快感を与えていく。構図内に男性が映る場合（特に顔や上半身が画面に入るシーン）は、必ず「顔が見えない（faceless male）」指定を行うこと。
