## 特徴

- **NSFW統合版**: SFWシーンの中に自然にNSFW要素が混ざり、シームレスな流れを実現
- **時系列変化**: 昼から夜へと時間が経過し、雰囲気も変化
- **段階的なNSFW**: 公園（Light）→親密な時間（Light+Moderate）→入浴（Light+Moderate）と徐々に進展
- **連番システム**: すべてのプロンプトに適切な連番が付与され、出力画像が時系列順に並ぶ

## カスタマイズ

- 男性タイプを変更したい場合: `male_type_default` を `male_type_dark_skin`、`male_type_shota` などに変更
- Sex場所を変更したい場合: `place_home_living_room` を他の場所に変更
- 時間帯を変更したい場合: 各シーンの `time_*` パラメータを調整
