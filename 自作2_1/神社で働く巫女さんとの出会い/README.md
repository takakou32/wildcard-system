# 神社で働く巫女さんとの出会い

## テーマ概要

神社で働く巫女さんとの出会いから親密な時間までを描くテーマです。

## シーンフロー

- **Scene 01: 昼** - 働いている巫女さん（SFWのみ）
- **Scene 02: 昼** - 巫女さんと合流（SFWのみ）
- **Scene 03: 夕方** - 神社内で巫女さんとイチャイチャ（SFW + NSFW Light）
- **Scene 04: 夜** - 親密な時間（NSFW準備）（SFW + NSFW Moderate）

## 設定

- **ベース**: lovey（ラブラブ）
- **テーマタイプ**: 神社・巫女さん系（特殊版）
- **男性タイプ**: デフォルト
- **時系列**: 昼 (time_day) → 夜 (time_night)
- **Sex場所**: place_shrine_office（神社事務所）
- **Sex服装**: miko（巫女服）

## NSFW統合設定

- **Scene 01**: NSFWなし
- **Scene 02**: NSFWなし
- **Scene 03**: NSFW Light（キス、ハグ、愛撫、胸タッチ）
- **Scene 04**: NSFW Moderate（手コキ、パイズリ、指マン、クンニ、フェラ軽め）
