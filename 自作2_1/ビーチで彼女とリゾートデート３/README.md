# ビーチで彼女とリゾートデート３

## テーマ概要

彼女とのビーチリゾートデートを午前中から夕暮れまで楽しむラブラブテーマです。
ビーチでの活動から、カフェでの休憩、夕暮れのロマンチックなひと時、そしてシャワーまで、一日の流れを自然に表現します。

## 基本情報

- **ベース**: lovey (ラブラブ)
- **テーマタイプ**: beach (ビーチ・リゾート系)
- **男性タイプ**: デフォルト (faceless male)
- **時系列**: late_morning → evening
- **Sex場所**: hotel_room (ホテル客室)

- 服装: カジュアルな服 (casual_day)

着替えやシャワーで一日の汗を流す

## ファイル構成

- `theme.txt` - テーマ名
- `config.yaml` - 設定情報（参考用）
- `pose_play.yaml` - Poseシーン定義（5シーン）
- `sex_play.yaml` - Sexシーン定義（6段階）
- `fellatio_play.yaml` - Fellatioシーン定義（8段階）
- `main.yaml` - 統合エントリーポイント
