# ビーチで彼女とリゾートデート2

## テーマ概要

彼女とのビーチリゾートデートを午後から夜まで楽しむラブラブテーマです。
ビーチでの活動から、カフェでの休憩、夕暮れのロマンチックなひと時、そしてホテルでの親密な時間まで、一日の流れを自然に表現します。

## 基本情報

- **ベース**: lovey (ラブラブ)
- **テーマタイプ**: beach (ビーチ・リゾート系)
- **男性タイプ**: デフォルト (faceless male)
- **時系列**: afternoon → night
- **Sex場所**: hotel_room (ホテル客室)

## シーンフロー

### Scene 01: afternoon - ビーチ到着・散策
ビーチに到着し、砂浜を歩いたり海を眺めたりする
- 服装: 水着 (swimsuit)
- 雰囲気: カジュアル

### Scene 02: afternoon - ビーチでの活動
水遊び、ビーチタオルでリラックス、日焼け止めを塗り合うなど
- 服装: 水着 (swimsuit)
- 雰囲気: カジュアル

### Scene 03: late_afternoon - ビーチカフェ
ビーチハウスやカフェで休憩、ドリンクや軽食を楽しむ
- 服装: カジュアルな服 (casual_day)
- 雰囲気: 心地よい

### Scene 04: evening - 夕暮れビーチ
夕日を眺めながらロマンチックな時間を過ごす
- 服装: 水着 (swimsuit)
- 雰囲気: ロマンチック

### Scene 05: evening - 更衣室・シャワー
着替えやシャワーで一日の汗を流す
- 服装: タオル (towel)
- 雰囲気: 心地よい

### Scene 06: night - リゾート客室
ホテルの部屋で二人きりの時間、窓からの景色を楽しむ
- 服装: 部屋着 (house_clothes)
- 雰囲気: ロマンチック
