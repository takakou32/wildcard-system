# プロンプト収集フロー マニュアル

Easy Reforge で生成した画像のプロンプトを `verified_scene_patterns.md` に保存するフロー。

---

## 基本的な使い方

Claude Code のチャット欄に入力するだけ。

```
/collect-prompts C:\work\EasyReforge__\stable-diffusion-webui-reForge\outputs\txt2img-images\2026-03-21
```

**日付フォルダのパスを変えるだけで毎回使える。**

---

## フローの流れ

```
1. 指定フォルダのPNG/WebPからプロンプトを自動抽出
2. キャラ外見・ライティング・場所・服装・LoRAを自動除去
3. シーンプロンプトを verified_scene_patterns.md のカテゴリに自動分類
4. 追加予定エントリ一覧を表示
5. 承認（y）したら一括追記 + TOC更新
```

ユーザー操作は**最後の承認のみ**。途中は全自動。

---

## 承認時の選択肢

| 入力 | 動作 |
|-----|------|
| `y` | 全件追加 |
| `n` | キャンセル |
| `2,3` など番号 | 指定エントリを除外して再確認 |

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `.claude/commands/collect-prompts.md` | スラッシュコマンドの定義（フロー指示） |
| `自作2_1/doc/scripts/extract_sd_prompts.py` | PNG/WebPメタデータ抽出スクリプト |
| `自作2_1/doc/verified_scene_patterns.md` | 保存先。冒頭のセクション一覧テーブルがID管理を兼ねる |

## よくある操作

### 今日生成した画像を収集する
```
/collect-prompts C:\work\EasyReforge__\stable-diffusion-webui-reForge\outputs\txt2img-images\2026-03-21
```

### 件数が多い場合（代表的な何件かだけ処理したい）
```
/collect-prompts C:\...\2026-03-21
```
→ サマリー表示後に除外番号を指定すればよい。
または「上位10件だけ対象にして」とClaudeに伝える。

### スクリプト単体でプロンプト確認だけしたい
```
python 自作2_1/doc/scripts/extract_sd_prompts.py "C:\...\2026-03-21" --limit 5
```

---

## 注意点

- Easy Reforge の出力形式（PNG / WebP）に両対応済み
- WebP は EXIF の ExifIFD > UserComment に UTF-16-BE で格納されているため専用の読み取り処理を使用
- LoRA タグ（`<lora:...>`）・キャラ名・アーティスト名・ライティング・場所は自動除去される
- `darkskin male` などの肌色指定もキャラ属性として除去し、scene pattern として汎用化される
